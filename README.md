\# 🚗 Smart Parking Reservation System



A Django-based web application that allows users to book parking slots efficiently with real-time availability, ML-based fee prediction, and QR code verification.



\---



\## 🔥 Features



\- User Registration \& Login

\- Book Parking Slots

\- Real-time Slot Availability

\- ML-based Fee Prediction

\- QR Code Generation

\- Booking History \& Cancel Option

\- User Profile with Avatar

\- Admin Dashboard with Analytics



\---



\## 🛠 Tech Stack



\- Python (Django)

\- Django REST Framework

\- SQLite

\- Scikit-learn (ML)

\- HTML, CSS, Bootstrap

\- Chart.js



\---
